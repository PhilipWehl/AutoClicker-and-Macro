## AutoClicker GUI 🖱️

A lightweight, customizable AutoClicker built with Python and Tkinter. This tool allows users to set a specific Click Per Second (CPS) rate and toggle the clicking action using a hotkey.

## ✨ Features

* **Customizable CPS:** Set your preferred clicking speed easily.
* Toggle the clicker on and off with a single keypress (default is F6).
* Low CPU and memory usage.

* And many more to come!
